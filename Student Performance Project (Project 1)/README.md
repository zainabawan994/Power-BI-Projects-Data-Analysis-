# Student Performance Data Analysis Dashboard

## Introduction

This project presents a data analysis dashboard built to analyze student performance using different factors such as study time, health, and social activities. The dashboard is created using Power BI and helps visualize how different variables affect students’ final marks.

The goal of this project is to perform exploratory data analysis and present insights using interactive visualizations.

---

## Dataset

The dataset used in this project is the Student Performance dataset. It contains information about students such as:

* Study time
* Health status
* Social activity (go out)
* Final marks
* Other academic and personal attributes

This dataset is useful for analyzing student behavior and academic performance.

---

## Dashboard Visualizations

The dashboard includes the following visualizations:

### 1. Count of Final Marks by Goout

This bar chart shows the number of students based on how often they go out. It helps analyze whether social activity affects student performance.

### 2. Health vs Final Marks (Scatter Plot)

This scatter plot shows the relationship between students’ health and their final marks. Each point represents a student. This visualization helps identify whether better health leads to better academic performance.

### 3. Study Time vs Final Marks

This chart shows the relationship between study time and final marks. It helps understand how study time impacts student performance.

## Project Objective

The main objectives of this project are:

* To analyze student academic performance

This dashboard helps in understanding how different factors such as health, study time, and social activities affect students’ final marks.
